## 🇸🇦 النسخة العربية (Arabic Version)

منصة برمجية متكاملة (Full-Stack) مدعومة بالذكاء الاصطناعي ومخصصة لمبرمجي تطبيقات الهواتف الذكية في العالم العربي الذين يستخدمون إطار العمل **Flutter** ولغة البرمجة **Dart**. يحاكي المشروع في وظيفته وتصميمه الأنظمة العالمية الرائدة مثل ChatGPT و Gemini بأسلوب زجاجي هادئ وبسيط مستوحى من واجهات Cortex.

---

### 🎨 أبرز المميزات البصرية وتجربة المستخدم
- **تصميم زجاجي عصري (Light Glassmorphism):** واجهة مستخدم ناصعة البياض مع تدرجات بنفسجية مريحة للعين مستوحاة من Stripe و Linear.
- **تناسق برمي فائق:** كتل برمجية كربونية معزولة تحتوي على أزرار محاكاة لنظام macOS (النقاط الثلاثية الملونة) مع حماية ألوان الخلفية.
- **مفتش التفكير لـ DeepSeek:** عرض معزول ومنسدل لخطوات التفكير التدريجي (Chain of Thought) للنموذج العميق بشكل أنيق.
- **شريط المقترحات التفاعلي القابل للتمرير:** شريط أفقي انسيابي يحتوي على 10 مقترحات برمجية عملية في Flutter، تقوم بملء حقل النص والتركيز عليه تلقائياً بمجرد نقرها.

---

### 🛠️ البنية والتقنيات المستخدمة

#### الواجهة الأمامية (React Client)
- **Vite & React 18:** لبناء واجهة سريعة وخفيفة وتحديث فوري للحالات.
- **Tailwind CSS:** لإدارة الأنماط البصرية والهوية الجمالية.
- **Framer Motion:** لإضافة تأثيرات بصرية وانتقالات انسيابية فائقة الدقة.
- **React Markdown & Highlight.js:** لعرض الأكواد المنسقة وتلوينها بدقة.

#### الواجهة الخلفية وقاعدة البيانات (Backend & DB)
- **Node.js & Express:** بناء خادم ويب خفيف ومستقل للـ REST APIs.
- **Sequelize ORM & SQLite:** قاعدة بيانات محلية آمنة ذاتية الإصلاح لتخزين المستخدمين وتاريخ المحادثات.
- **JWT & Bcrypt:** لتأمين وتشفير كلمات المرور والتحقق من صلاحية الجلسات.
- **Hugging Face / OpenAI Router:** استدعاء النماذج الذكية بأمان مع ميزة موازنة الضغط والتوجيه التلقائي للمخدم الأسرع.

### 📂 الهيكل التنظيمي للمشروع

```plaintext
/Flutter-AI-Mentor
│
├── package.json               # ملف الإدارة الرئيسي لتشغيل وإعداد مستودع الـ Monorepo
│
├── /backend                    # خادم الـ Express وقاعدة البيانات
│   ├── .env.example            # قالب ملف الإعدادات السرية
│   ├── server.js               # نقطة الانطلاق الرئيسية للسيرفر
│   ├── db.js                   # ملف الموديلات والعلاقات وقاعدة البيانات الموحد
│   └── /routes                 # مسارات التحكم البرمجي (auth.js, ai.js)
│
└── /frontend                   # كود الواجهة الأمامية لـ React
    ├── src/
    │   ├── App.jsx             # منسق الواجهة وموجه الصفحات الرئيسي
    │   ├── index.css           # ملف الأنماط البصرية المطور بكسل بكسل
    │   └── main.jsx            # ملف ربط وحقن الـ DOM لـ React
🚀 طريقة التثبيت والتشغيل المحلي (من الصفر)
المتطلبات الأساسية
تأكد من تثبيت بيئة تشغيل Node.js (v18+) على جهازك.
1️⃣ تحميل المشروع وتثبيت الحزم
افتح الـ Terminal في جهازك، وقم بنسخ المستودع وتثبيت كافة الحزم والتبعيات في كل المجلدات بضغطة زر واحدة:
code
Bash
# نسخ المشروع محلياً
git clone https://github.com/YOUR_USERNAME/flutter-hub.git
cd flutter-hub

# تثبيت الحزم لكافة طبقات المشروع بشكل متزامن وآمن لجميع الأنظمة
npm run install:all
2️⃣ إعداد المفاتيح السرية والـ API Keys
اذهب إلى مجلد الـ backend.
قم بأخذ نسخة من ملف .env.example وتسميتها بـ .env:
code
Bash
cp backend/.env.example backend/.env
افتح ملف backend/.env واكتب مفاتيحك الخاصة فيه:
ضع التوكن الخاص بك في خانة HF_TOKEN.
ضع رمز تشفير قوي لـ JWT_SECRET.
3️⃣ تشغيل بيئة التطوير بضغطة زر
عد إلى المجلد الرئيسي للمشروع (الجذر) واكتب هذا الأمر لتشغيل السيرفر والفرونت أند معاً في نفس الوقت:
code
Bash
npm run dev
سيقوم هذا الأمر بتشغيل سيرفر Express على المنفذ (5000) وسيرفر Vite React على المنفذ (5173) بشكل متزامن.
🔒 الأمان وخصوصية البيانات
حماية المفاتيح السحابية: جميع المفاتيح والـ API keys مشفرة محلياً داخل ملف backend/.env ومحجوبة تماماً عن الرفع لـ GitHub بواسطة ملف الـ .gitignore.
خصوصية جلسات العمل: قاعدة البيانات بالكامل database.sqlite تعمل وتخزن محلياً داخل جهازك لضمان عدم اطلاع أي جهات خارجية على بياناتك ومحادثاتك.
